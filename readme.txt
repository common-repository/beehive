Beehive plugin makes your post or pages as gallery, - easy accessible from frontend like a beehive in hexagonal cell from for each post or page featured image

== Description ==

Goal of this awsome project is to make your site/page outstand from other sites.
Beehive plugin will enable a widget that can get your latest posts or pages featured images and make them to some cool accessible cell links.

== Installation ==

1. Upload `beehive` plugin to the `/wp-content/plugins/` directory manually or via WordPress plugin menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set one beehive widget in the correct place and choose if it should take posts or pages

== Frequently Asked Questions ==

= How can I set up beehive for a custom post type =

Get our beehive pro plugin and you will support also maintenance of this plugin too for newer versions of wordpress.

= What is the animation about =

Animation is for the demo purpose but It's also available in the pro version of the plugin.

= My other beehive instance is breaking on the same page =

You are allowed to use only one beehive per page but on the beehive pro version of the plugin you can use as many as you need.
